# DevOps Kubernetes CI/CD Pipeline

## 🚀 Overview

This project demonstrates an end-to-end CI/CD pipeline using GitHub Actions, Docker, and Kubernetes.

## 🧰 Tech Stack

* Docker
* Kubernetes
* GitHub Actions
* Flask
* Linux

## ⚙️ CI/CD Workflow

1. Code pushed to GitHub
2. GitHub Actions builds Docker image
3. Image pushed to Docker Hub
4. Kubernetes deploys application

## 🐞 Issues Resolved

* CrashLoopBackOff
* ErrImagePull
* Container startup failures

## 📊 Outcome

* Automated deployment pipeline
* Improved deployment reliability
